# Features Documentation

This directory contains user-facing documentation for SakaiBot features.

## Available Features

### Image Generation
- **[image-generation.md](image-generation.md)** - Text-to-image generation using Flux and SDXL models

## Documentation Format

Each feature document includes:
- Overview and purpose
- User commands and examples
- Configuration requirements
- Feature capabilities
- Error handling
- Limitations

## Target Audience

These documents are written for:
- End users who want to use the features
- Administrators who need to configure features
- Developers who need to understand feature behavior
