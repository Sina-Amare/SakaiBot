# Architecture Documentation

This directory contains system architecture and design documentation.

## Available Documentation

### Image Generation
- **[image-generation-architecture.md](image-generation-architecture.md)** - Complete architecture of the image generation system

## Documentation Format

Each architecture document includes:
- System overview
- Component architecture
- Data flow diagrams
- Design decisions and rationale
- Configuration integration
- Error handling architecture
- Testing architecture
- Security considerations
- Performance considerations
- Future extensibility

## Target Audience

These documents are written for:
- Developers who need to understand system design
- Architects who need to review design decisions
- Maintainers who need to extend or modify the system
